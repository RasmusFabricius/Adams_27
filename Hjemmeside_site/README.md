# Adams Fødselsdag - Gåder

En simpel statisk hjemmeside med gåder til Adam.

Sådan kører du den lokalt:

1. Åbn Finder og gå til mappen `Hjemmeside_site`.
2. Dobbeltklik `index.html` for at åbne i din browser (eller højreklik -> Åbn med -> Safari/Chrome).

Skriv svarene i inputfeltet (svarene er på dansk). Rigtige svar viser confetti; forkerte svar udløser gulerødder.
Rigtige svar viser nu ænder (🦆); forkerte svar udløser gulerødder.

Der er tilføjet flere gåder, inklusiv nogle med Disney-tema (fx Mickey, Askepot/Cinderella og Simba).

Der er også billed-gåder (Street View / foto). For at de virker skal du placere billeder i mappen `sv/` i projektroden med følgende navne:

- `sv/rhodos.jpg`
- `sv/gdansk.jpg`
- `sv/monsted.jpg`
- `sv/grand_canyon.jpg`
- `sv/pando.jpg`
- `sv/new_york.jpg`
- `sv/japan.jpg`
- `sv/milano.jpg`

Du kan hente billedeudsnit fra Google Street View manuelt eller bruge egne fotos. Når filerne er i `sv/` vil de automatisk vises i gåden og Adam skal gætte stedet.

Gåderne kan redigeres i `script.js`.
